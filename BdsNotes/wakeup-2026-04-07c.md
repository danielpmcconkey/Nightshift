# Wakeup — 2026-04-07c

## What happened this session

### File-based agent communication (Ogre pattern port)

Ported the interstitial file pattern from EtlReverseEngineering to Nightshift. The engine no longer parses agent stdout for routing decisions. Agents and the Foreman now write their outcomes to process artifact files; the engine reads those files. Stdout is saved as `.stdout.log` for debugging only.

This kills the markdown fence problem that caused false auto-escalations on P022 and P056. The engine doesn't care what format Claude wraps its conversational output in anymore.

**Changed files:**
- `OutcomeParser.cs` — gutted stdout fallback, file-only reads, added `ParseForemanArtifact()`
- `AgentInvoker.cs` — switched to `--output-format text`, foreman reads artifact file, killed `ParseForemanResponse()`
- `PromptBuilder.cs` — "Response Contract" sections with explicit file paths, foreman gets artifact path param
- `ArtifactManager.cs` — added `GetForemanArtifactPath()` (`_foreman_{step}.json`)
- `StepHandler.cs` — stdout goes to `.stdout.log`, not the process artifact
- `blueprints/foreman-jurisdiction.md` — response format rewritten for file-write contract

**Commit:** `d921139`

### Card dependencies

Added card dependency support. Cards can declare blocking dependencies on other cards (many-to-many, not hierarchy). The engine skips cards whose dependencies aren't all complete.

**Changed files:**
- `sql/003_card_dependencies.sql` — new junction table with composite PK, no self-ref CHECK
- `TaskQueueRepository.cs` — claim query has `NOT EXISTS` for incomplete dependencies
- `CardRepository.cs` — `AddDependency`, `RemoveDependency`, `GetDependencies`, `GetDependencyStatuses`

**Commit:** `4311ef0`

Migration has been applied to the nightshift DB.

### Design decisions made

- Dependencies are blocking relationships, not parent-child hierarchy. Epics (containment) will be a separate concept later.
- Card metadata (title, description, priority, dependencies) is immutable once `status != 'queued'`. Not enforced in code yet — will be enforced at the management surface layer when built.
