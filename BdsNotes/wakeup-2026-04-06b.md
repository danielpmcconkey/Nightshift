### Key improvements from deepening

- **Per-gate conditional counts** (JSONB) — fixes a 9-loop bug where the flat
  counter let a card bounce 3x at reviewer + 3x at governor + 3x more
- **CliWrap** for subprocess management — prevents .NET stdout/stderr deadlock
- **Generic Host + BackgroundService** — graceful shutdown with CancellationToken
- **NpgsqlDataSource** — modern pooled API, not raw connections
- **Security hardening**: ProcessStartInfo.ArgumentList (not string), env var
  whitelist for child processes, Serilog secret redaction, path validation
- **Explicit transaction boundaries** — StepHandler owns transactions
- **FK indexes** on all FK columns (Postgres doesn't auto-create them)
- **Richer Foreman outcomes** — `inject_context` field for feeding context forward
- **Per-card scratchpad** (`notes.md`) for inter-agent knowledge sharing
- **Unknown outcomes route to Foreman** instead of auto-failing
- **Smoke test** with stub agent defined concretely in Phase 7
